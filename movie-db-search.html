In a team of three, we managed to create a Business Brochure website. Check out the wireframe and the mockup of the website by clicking on the links. This is a fictional resturant which serve varieties of food. The website was created using HTML, SASS, JavaScript and version control in Git. Wireframes and Mockup were put together with Balsamiq and Invision respecttively. The project was introduced after two weeks of learning the Technical Web Designer Program at BCIT.
