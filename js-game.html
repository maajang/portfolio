Tools Used

This project is demo for Javascript game development capability. The game is simple but interesting with few fancy functionalities implemented. It was as result of mixing up HTML, CSS and JavaScript. The project was introduced two and half weeks after the program started. I intend to improve the game by adding scoreboard and maybe make a splash page prior to the game page to make it even more fancy. Also, I have been involved in a Javascript Game development prior to this in a team of three at UBC which you can find . The tools were OpenGL, Three.js and glsl (shading language for OpenGL)
